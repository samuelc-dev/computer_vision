Cofnigurações necessárias para a instalação e execução;

    Liguagem de programação e pacotes:
        - python3
        - Gerenciador de pacotes de projeto "PIP":
                obs: Verifique se já possui o python instalado no seu SO.
                Para instalar o "PIP" no linux Ubumtu ou derivados: sudo apt-get install pip
                Verifique a versão do "PIP": pip --version
    
    Bibliotecas necessárias:
        - OpenCV:
            obs: Antes de iniciar a instalação do OpenCV, é importante lembrar que algumas dependências devem estar instaladas no sistema. Entre elas estão o NumPy, Matplotlib e o SciPy. Verifique se essas bibliotecas estão instaladas antes de prosseguir.
                * Instalação das dependências:
                    - NumPy: pip install numpy
                    - Matplotlib: pip install matplotlib
                    - SciPy: pip install SciPy
                * Instalação do OpenCV:
                    - Para instalar o OpenCV digite o comando: pip install opencv-python
        - Cvzone:
            * Instalação do CVzone:
                - Para instalar o CVzone digite o comando: pip install cvzone
        - MediaPipe:
            * Instalação do MediaPipe:
                - Para instalar o MediaPipe digite o comando: pip install mediapipe
        - Pynput
            * Instalação do Pynput:
                - Para instalar o Pynput digite o comando: pip install pynput
